# 01_html面试题

# 01_行内元素有哪些？块级元素有哪些？空元素有哪些？

  说出代表性几个就行
  行内元素 span、img、input...... a、b、strong、select、label、em、i 、button、textarea
  块级元素 div、footer、header、section、h1-h6...... ul、li、dl、dt、dd、p、blockquote 
  
  面试官可能峰回路转，问你元素之间块与行内的转换
  display: inline;  			把某元素转换成了行内元素 -> 不独占一行的，并且设置宽高是无效的，宽高随文字（行内元素的内容）撑开
  display: inline-block;  把某元素转换成了行内块元素 -> 不独占一行，并且可以设置宽高
  display: block; 				把某元素转换成了块元素 -> 独占一行，可以设置宽高

# 02_页面导入样式时，使用link和@import的区别?

区别一：link是先有的，后有@import (兼容性上link比@import好,体现在ie5);
区别二：加载顺序的差别，浏览器先加载link标签，后加载@import

# 03_title与h1的区别？b与strong的区别？i与em的区别？

title与h1的区别：

定义：
  <title>植发医院_植发价格_植发多少钱发友交流_植发吧</title> 
	title：概括了网站信息，可以告诉搜索引擎或者用户关于这个网站的主题内容是什么
	sohu.com的所有文章的标题都是h1
	h1：文章主题内容，告诉蜘蛛我们的网站内容最主要的是什么
区别：
	title它是显示在网页标题上，h1是显示在网页内容上
	title比h1更加的重要 （title > h1） -> 对于seo的了解
场景：
	网站的logo都是用h1标签包裹的

b与strong的区别:

定义：
	b：实体标签，用来给文字加粗的
	strong：逻辑标签，用来加强字符语气的
区别：
	b标签只有加粗的样式，没有实际的含义，没有实际的语义
	strong表示标签内的字符比较重要，用于强调的
题外话：为了符合css3规范，b尽量少用，多用strong就行了

i与em的区别：

定义：
	i：实体标签，用来做文字倾斜的
	em：是逻辑标签，用来强调文字内容的
区别：
	i只是一个倾斜标签，没有实际含义
	em标签标签内字符重要，用于强调的
场景：
	i更多的用在字体图标
	em更多用在术语上（医药，生物）

# 04_img标签的title和alt有什么区别?

<img src="hero.png" title="这是一张图片" alt="这是alt">

区别一：
  title : 鼠标移入图片显示的值
  alt   : 图片无法加载时显示的值
区别二：
	在seo的层面上，蜘蛛抓取不到图片的内容，所以前端在写img标签的时候为了增加seo效果要加入alt属性来描述这张图是什么内容或者关键词

# 05_png、jpg、gif这些图片格式解释一下，分别什么时候用?

png: 无损压缩，尺寸体积比jpg/jpeg要大,适合做小图标
jpg: 采用压缩算法，有一点失真，比png体积要小,适合做中大图
gif: 一般做动态图的，很少用
webp：同时支持有损或者无损压缩，相同质量的图片，webp具有更小的体积，兼容性不是特别好（Chrome可以，IE不行....）,webp在小程序模拟器上可以正常显示，到了真机调试的时候，webp格式的图片是出不来的
